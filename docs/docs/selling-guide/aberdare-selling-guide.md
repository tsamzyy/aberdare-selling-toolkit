# Aberdare Selling Guide (CF44)

## 1) Price it right
- Ask for a data-backed valuation (recent sold prices + current competition)
- Be realistic about condition, EPC, parking, and location within CF44

## 2) Presentation that sells
- Declutter, deep clean, fix obvious snags
- Bright photos, tidy garden/entrance, neutral rooms

## 3) Choose an agent who reduces fall-throughs
- Clear communication plan
- Proactive chain management
- Strong buyer qualification

## 4) Documents to prepare early
- ID and proof of address
- Title info (if available)
- EPC (or book it)
- FENSA/warranties/guarantees
- Leasehold info (if relevant)

## 5) Accepting an offer without pain later
- Confirm buyer status (mortgage agreed? deposit?)
- Agree target timescales
- Book solicitor early
